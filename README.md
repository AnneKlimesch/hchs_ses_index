# hchs_ses_index
This repository holds code for developing the socioeconomic status (SES)-Index for the Hamburg City Health Study (HCHS).

"hchs_data_preparation.Rmd" prepares the HCHS dataset for calculating the SES-Index. 

"hchs_ses_index.Rmd" (to be added soon) can be used to determine the SES-Index in the HCHS dataset but can also be adapted to determine the SES-Index for other datasets.
